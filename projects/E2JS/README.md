# E2JS
Simple Excel VBA macro that lets you select cells and copy them to the clipboard as a Javascript array (of arrays).

## About
E2JS formats a selection of cells as a JavaScript array. The array is then available via the clipboard. E2JS is bound to the shortcut ctrl+shift+j.

## Install Instructions
1) File -> Save As
2) Change file type to Excel Add In (.xlam)
3) Make sure you the folder you are saving in ends in AppData\Roaming\Microsoft\AddIns
4) Use with ctrl+shift+j
