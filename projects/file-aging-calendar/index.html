Overview

About

The graph above is a visual representation of the last modified date of files containing sensitive information. Each row represents a year with large boxes representing months and small cells representing days.

The cells are color coded as a heatmap. White cells show days in which minimal files were modified. Red cells show days in which large amounts of files were modified.

Hovering over a cell will display a tooltip showing the number of files last modified on that day.

This graph is also responsive and will remain functional on any kind size of screen - try resizing the browser window and watch it change.
